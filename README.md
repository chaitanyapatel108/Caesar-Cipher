# Caesar Cipher

A simple encryption and decryption program implemented in C++.  
It allows users to securely encode and decode messages using a Caesar cipher algorithm,  
with password protection for each saved message.

## Features
- Encode and decode text using Caesar cipher logic  
- Password-protected message storage  
- Save, delete, or clear messages easily  
- Bulk and single message processing
